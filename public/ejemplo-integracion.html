Integración del Sistema de Reservaciones

Opción 1: Integración con JavaScript (Recomendada)

Agrega este código HTML donde quieras mostrar el sistema de reservaciones:

<div id="sistema-reservaciones-container"></div>

<script src="https://sistema-reservaciones.onrender.com/embed.js"></script>
<script>
  document.addEventListener('DOMContentLoaded', function() {
    iniciarSistemaReservaciones('sistema-reservaciones-container', {
      restaurantName: "CHIPOTLE MEXICAN GRILL",
      primaryColor: "#0d925f",
      buffetColor: "#fd9b17",
      height: "800px"
    });
  });
</script>

Opciones de Personalización

Puedes personalizar el sistema con estas opciones:

  • restaurantName: Nombre del restaurante
  • primaryColor: Color principal para botones (por defecto "#0d925f")
  • buffetColor: Color para opciones de buffet (por defecto "#fd9b17")
  • showBuffet: Mostrar opción de buffet dominical (por defecto true)
  • width: Ancho del sistema (por defecto "100%")
  • height: Alto del sistema (por defecto "800px")

Opción 2: Usando iframe

Si prefieres usar un iframe directamente:

<iframe 
  src="https://sistema-reservaciones.onrender.com?embedded=true&restaurantName=CHIPOTLE%20MEXICAN%20GRILL" 
  style="width: 100%; height: 800px; border: none; border-radius: 8px; box-shadow: 0 4px 8px rgba(0,0,0,0.1);"
></iframe>
